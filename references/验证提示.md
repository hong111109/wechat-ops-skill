# Validation Prompts

## Prompt 1：起号定位
**Prompt:** 我想做一个公众号，但不知道该写给谁看。
**Expected route:** `01-定位与战略.md`
**Expected mode:** 顾问模式
**Expected output signals:** 给出定位判断、读者画像、内容支柱、不写清单

## Prompt 2：栏目系统
**Prompt:** 帮我直接出一个适合个人IP的一周选题和栏目安排。
**Expected route:** `02-选题与栏目系统.md`
**Expected mode:** 操盘模式
**Expected output signals:** 给栏目矩阵、选题池、发布节奏、主目标标注

## Prompt 3：文章改稿
**Prompt:** 这是我的标题和开头，帮我改得更容易被点开和读下去。
**Expected route:** `03-写作与包装.md`
**Expected mode:** 教练模式
**Expected output signals:** 先判断问题，再给标题备选、开头改写、结构建议、CTA建议

## Prompt 4：增长问题
**Prompt:** 为什么我的文章总是阅读量不高？
**Expected route:** `04-增长与分发.md`
**Expected mode:** 顾问模式
**Expected output signals:** 不直接怪文笔，先拆成选题、标题、分发、目标读者几个层次

## Prompt 5：私域承接
**Prompt:** 帮我设计公众号到私域咨询的承接链路。
**Expected route:** `05-转化与诊断.md`
**Expected mode:** 操盘模式
**Expected output signals:** 给出链路、CTA、承接动作、观察指标

## Prompt 6：数据诊断
**Prompt:** 阅读量还行，但几乎没人加我，问题可能在哪？
**Expected route:** `05-转化与诊断.md`
**Expected mode:** 诊断模式
**Expected output signals:** 先判断问题更可能出在 CTA、承接或 offer，而不是继续堆流量

## Prompt 7：小团队协作
**Prompt:** 我们三个人做公众号，怎么分工和开周会？
**Expected route:** `06-轻协作与反模式.md`
**Expected mode:** 操盘模式
**Expected output signals:** 给角色分工、周节奏、轻量 SOP

## Prompt 8：系统问题伪装成单篇问题
**Prompt:** 我这篇文章怎么改才能更火？
**Expected route:** `03-写作与包装.md` + `04-增长与分发.md`
**Expected mode:** 先诊断再教练
**Expected output signals:** 先判断这是单篇问题还是定位 / 分发问题，不直接只改句子
